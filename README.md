# Classroom Documentation Tracker
This project is a Python-based tool designed to track work across Montessori education disciplines. 
## Features
- Record total student activity and weekly breakdowns
- Track work across educational disciplines by 
- Generate summaries by discipline
- Analyze individual student activity
- Identify missing areas of engagement for each child
## Purpose
This tool was created to ensure the students experienced a balanced range of activities across all areas of the classroom and to provide insight into the interests of each child.
