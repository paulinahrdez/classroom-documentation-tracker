# Classroom Documentation Tracker
This project is a Python-based tool for tracking work across Montessori educational disciplines. 
## Features
- Record total student activity and weekly breakdowns
- Generate summaries by discipline
- Analyze individual student activity
- Identify missing areas of engagement for each child
## Purpose
This tool was created to ensure students experienced a balanced range of activities across all areas of the classroom and to provide insight into each child's interests.
